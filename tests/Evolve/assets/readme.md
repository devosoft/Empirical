# Contents of this directory

- hang_test.bad: A phylogeny for use in the "Test LoadFromFile MPD Hang" regression test. This phylogeny previously caused calculating mean pairwise distance to hang after being loaded via LoadFromFile, due to a discrepancy in how extant taxa were inferred. Details: https://github.com/emilydolson/phylotrackpy/issues/87.
- nk_ecoeaselection.csv: Example phylogeny generated using an NK landscape and eco-ea
- nk_lexicaseselection.csv: Example phylogeny generated using an NK landscape and lexicase selection
- nk_tournamentselection.csv: Example phylogeny generated using an NK landscape and tournament selection

Base class for all sizes of Othello games.

  • Hide almost everything. Only give users access to game-advancing functions (don‚Äôt allow willy-nilly board manipulation, etc). This would let us make lots of assumptions about calculating flip lists, scores, etc, which would speed up asking for flip lists, etc. multiple times during a turn.
